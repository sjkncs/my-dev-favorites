# 2026-01-29~02-04 本周arXiv前沿论文

### 1. Scaling Embeddings Outperforms Scaling Experts
- **arXiv**: 2601.21204
- **摘要**: 证明在MoE架构中扩展嵌入维度优于增加专家数量，推出LongCat-Flash-Lite（68.5B总参/~3B激活），在编码和Agent基准上表现强劲
- **类别**: LLM理论 / 架构

### 2. LatentLens: Revealing Interpretable Visual Tokens in LLMs
- **arXiv**: 2602.00462
- **摘要**: 将多模态LLM的隐藏视觉状态解码为可解释文本，发现视觉Token携带远超标准指标衡量的语义信息
- **类别**: LLM理论 / 多模态

### 3. ERNIE 5.0: 2.4T Parameter Unified Multimodal Foundation
- **arXiv**: 2602.04705
- **摘要**: 百度发布2.4万亿参数统一多模态基座模型（文本+图像+音频），首个生产级万亿参数自回归模型，弹性训练+超稀疏MoE
- **类别**: 大模型 / 多模态

### 4. State-Transition Framework for Efficient LLM Reasoning
- **arXiv**: 2602.01198
- **摘要**: 将LLM推理建模为状态转移序列，线性注意力访问历史上下文避免二次成本，状态驱动门控减少错误中间步骤引发的过度思考
- **类别**: 推理 / 效率

### 5. PrAg-PO: Prompt Augmentation Scales up GRPO Training
- **arXiv**: 2602.03190
- **摘要**: 提示模板增强+模板特定格式奖励，提升GRPO训练多样性和防止早期训练崩溃，在数学推理RL微调中效果显著

### 6. IntentCoding: Amplifying User Intent in Code Generation
