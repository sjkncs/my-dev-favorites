# 2026-07-23 本周 arXiv 前沿论文

### 1. Off-Context GRPO: Learning to Reason on Hard Problems using Privileged Information
- **机构**: arXiv (cs.LG)
- **摘要**: 提出离上下文 GRPO，在训练时引入"特权信息"帮助模型攻克难题的推理链，推理阶段无需该信息，缓解难样本上的强化学习稀疏奖励问题。
- **Link**: [arXiv:2607.19313](https://arxiv.org/abs/2607.19313)

### 2. ISO: An RLVR-Native Optimization Stack
- **机构**: arXiv (cs.LG)
- **摘要**: 面向"可验证奖励强化学习"（RLVR）从底层重构的优化栈，将奖励验证、采样与梯度更新协同设计，提升 RLVR 训练的稳定性与吞吐。
- **Link**: [arXiv:2607.19331](https://arxiv.org/abs/2607.19331)

### 3. Adopting Reinforcement Learning with Verifiable Rewards for Molecular Generation
- **机构**: arXiv (cs.LG)
- **摘要**: 将 RLVR 迁移到分子生成任务，用可验证的化学性质作为奖励，指导生成满足多目标约束的候选分子，拓展 RLVR 在科学发现的应用边界。
- **Link**: [arXiv:2607.19044](https://arxiv.org/abs/2607.19044)

### 4. AdaFlash: Adaptive Speculative Decoding via On-Policy Distilled Diffusion Drafters
- **机构**: arXiv (cs.LG)
- **摘要**: 用在线策略蒸馏的扩散草稿模型实现自适应推测解码，根据难度动态调整草稿长度，在保持质量的同时显著加速 LLM 推理。
- **Link**: [arXiv:2607.19223](https://arxiv.org/abs/2607.19223)

### 5. Where Should Optimizer State Live? Tiered State Allocation for Memory-Efficient MoE Training
- **机构**: arXiv (cs.LG)
- **摘要**: 研究 MoE 训练中优化器状态的分层放置策略（显存/主存/磁盘分级），在有限显存下训练更大专家模型，降低训练成本。
- **Link**: [arXiv:2607.19058](https://arxiv.org/abs/2607.19058)

### 6. H²SD: Hybrid Hindsight Self-Distillation
- **机构**: arXiv (cs.LG)
- **摘要**: 结合"事后经验"信号的混合自蒸馏方法，让模型从失败与成功轨迹中同时学习，提升推理与决策能力。
- **Link**: [arXiv:2607.18955](https://arxiv.org/abs/2607.18955)

### 7. Toward Auditable Fraud Detection: Combining Graph Features, Model Explanations, and Agentic Case Investigation
- **机构**: arXiv (cs.LG)
- **摘要**: 将图特征、模型可解释性与"智能体式案件调查"工作流结合，构建可审计的欺诈检测系统，兼顾检出率与合规解释性。
- **Link**: [arXiv:2607.19266](https://arxiv.org/abs/2607.19266)

### 8. From Trajectories to Instructions: Language-Conditioned Meta-Reinforcement Learning
