# 2026-06-13 本周arXiv前沿论文

## 1. PP-OCRv6: From 1.5M to 34.5M Parameters, Surpassing Billion-Scale VLMs
- **机构**: 百度 PaddlePaddle 团队
- **摘要**: 轻量级OCR系统，仅用34.5M参数即超越十亿级多模态大模型的OCR能力。在精度和效率之间实现了极佳平衡，适用于移动端和边缘部署场景。
- **Link**: [arXiv:2606.13108](https://arxiv.org/abs/2606.13108)
- **重要性**: 高 — 证明了小模型在特定任务上完全可以超越大模型

## 2. SENTINEL: Failure-Driven Reinforcement Learning for Tool-Using LLM Agents
- **作者**: Ziyi Wang, Yuxuan Lu 等
- **摘要**: 提出失败驱动强化学习框架，高效训练工具使用型LLM Agent。通过从失败案例中学习，显著提升agent在复杂工具调用场景中的成功率。
- **Link**: [arXiv:2606.12908](https://arxiv.org/abs/2606.12908)
- **重要性**: 高 — 为工具使用型agent的训练提供了新范式

## 3. Boltzmann Attention: Learnable Ising Couplings for Cooperative Attention
- **作者**: Gilhan Kim, Daniel K. Park
- **摘要**: 将统计物理中的Ising模型引入注意力机制，提出可学习的Ising耦合注意力。为注意力机制提供了全新的数学视角，可能在多模态融合和复杂推理中展现优势。
- **Link**: [arXiv:2606.12478](https://arxiv.org/abs/2606.12478)
- **重要性**: 中 — 跨学科创新，长期潜力大

## 4. HYDRA-X: Native Unified Multimodal Models with Holistic Visual Tokenizers
- **作者**: Zhang, Qiu, Cui 等
- **摘要**: 提出统一多模态模型架构，采用整体式视觉tokenizer，实现图像、视频、文本的原生统一表示。在多项多模态基准上取得领先性能。
- **Link**: [arXiv:2606.13289](https://arxiv.org/abs/2606.13289)
- **重要性**: 高 — 多模态统一架构的重要进展

## 5. The Hidden Power of Scaling Factor in LoRA Optimization
- **作者**: Zicheng Zhang, Haoran Li 等
- **摘要**: 揭示LoRA缩放因子alpha的独立影响，发现alpha的调节对微调效果具有决定性作用。为LoRA参数调优提供了理论指导和实证分析。
- **Link**: [arXiv:2606.12883](https://arxiv.org/abs/2606.12883)
- **重要性**: 中 — 对实际工程调参有直接指导意义

## 6. ExpWeaver: LLM Agents Learn from Experience via Latent RAG
- **摘要**: 通过将经验编码为潜在表示，实现LLM Agent的经验积累和复用。采用Latent RAG机制，让agent能够从过往交互中持续学习，提升长期任务执行能力。
- **Link**: [arXiv:2606.01041](https://arxiv.org/abs/2606.01041)
- **重要性**: 中 — Agent记忆和学习机制的创新

## 7. REFLECT: Intervention-Supported Error Attribution for Silent Failures in LLM Agents
- **摘要**: 针对LLM Agent中的"静默失败"问题，提出干预支持的错误归因框架。能够识别和定位agent执行过程中的隐性错误，提升系统可靠性。
- **Link**: [arXiv:2606.09071](https://arxiv.org/abs/2606.09071)
- **重要性**: 中 — Agent可靠性工程的重要工具
