# 2026-07-20 本周arXiv前沿论文

### 1. Self-GC: Self-Governing Context for Long-Horizon LLM Agents
- **机构**: arXiv (cs.AI)
- **方向**: Agent
- **摘要**: 面向长程任务的 LLM 智能体上下文自治机制，让 Agent 自主管理、压缩与治理上下文，缓解长时序任务中的上下文膨胀与遗忘。
- **Link**: [arXiv:2607.00692](https://arxiv.org/abs/2607.00692)

### 2. Can Agents Generalize to the Open World? Unveiling the Fragility of Static Training in Tool Use
- **机构**: ICML 2026
- **方向**: Agent
- **摘要**: 揭示静态训练的工具使用智能体在开放世界中的脆弱性，指出泛化能力不足是当前工具调用 Agent 的核心短板。
- **Link**: [arXiv:2607.01084](https://arxiv.org/abs/2607.01084)

### 3. Self-Evolving Agents with Anytime-Valid Certificates
- **机构**: arXiv (cs.AI)
- **方向**: Agent
- **摘要**: 在自我进化的同时提供 anytime-valid 统计有效性证书，让智能体的持续改进带有可验证的置信保证。
- **Link**: [arXiv:2607.00871](https://arxiv.org/abs/2607.00871)

### 4. AutoMem: Automated Learning of Memory as a Cognitive Skill
- **机构**: arXiv (cs.AI)
- **方向**: Agent
- **摘要**: 把“记忆”当作一种可学习的认知技能进行自动化学习，而非静态外挂检索，提升智能体的长期记忆与调用效率。
- **Link**: [arXiv:2607.01224](https://arxiv.org/abs/2607.01224)

### 5. Revisiting Chain-of-Thought Reasoning under Limited Supervision
- **机构**: arXiv (cs.LG)
- **方向**: Reasoning
- **摘要**: 在有限监督下重新审视思维链推理，提出半监督方法降低对高质量 CoT 标注的依赖。
- **Link**: [arXiv:2607.01511](https://arxiv.org/abs/2607.01511)

### 6. Beyond Next-Token Prediction: An RLVR Proof of Concept for Tool-Use Agents on Atlassian Workflows
- **机构**: arXiv (cs.LG)
- **方向**: RL
- **摘要**: 将可验证奖励强化学习（RLVR）应用于 Atlassian 工作流的工具使用智能体，超越下一 token 预测范式。
- **Link**: [arXiv:2607.01465](https://arxiv.org/abs/2607.01465)

### 7. The Wiola Architecture for Efficient Small Language Models
- **机构**: arXiv (cs.LG)
- **方向**: Efficiency
- **摘要**: 提出面向高效小语言模型（SLM）的 Wiola 架构，在端侧/低算力场景下兼顾质量与效率。
- **Link**: [arXiv:2607.01394](https://arxiv.org/abs/2607.01394)

### 8. Hawk: Harnessing Hardware-Aware Knowledge for High-Performance NPU Kernel Generation
- **机构**: arXiv (cs.LG)
- **方向**: Efficiency
- **摘要**: 利用硬件感知知识自动生成高性能 NPU kernel，面向端侧推理加速与国产算力适配。
- **Link**: [arXiv:2607.01590](https://arxiv.org/abs/2607.01590)
