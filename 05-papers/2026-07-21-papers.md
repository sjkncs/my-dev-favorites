# 2026-07-21 本周 arXiv 前沿论文

> 采样自 arXiv cs.LG / cs.AI 最新提交（2026-07-20 列表），聚焦大模型、智能体、推理、强化学习、效率与多模态方向。

### 1. PagedWeight: Efficient MoE LLM Serving with Dynamic Quality-Aware Weight Quantization
- **方向**: Efficiency / MoE 推理服务
- **摘要**: 提出面向 MoE 大模型的服务方法，按质量感知动态量化权重，在保证输出质量的前提下显著降低显存与服务成本。
- **Link**: [arXiv:2607.16184](https://arxiv.org/abs/2607.16184)

### 2. When Does Muon Help Agentic Reinforcement Learning?
- **方向**: RL / Agent
- **摘要**: 系统研究 Muon 优化器在智能体强化学习中的适用边界，给出"何时有效、何时失效"的经验判据。
- **Link**: [arXiv:2607.16169](https://arxiv.org/abs/2607.16169)

### 3. When Do Multi-Agent Systems Help? An Information Bottleneck Perspective
- **方向**: Multi-Agent / 理论
- **摘要**: 从信息瓶颈理论视角分析多智能体系统的收益来源与失效条件，为"是否值得上多智能体"提供理论依据。
- **Link**: [arXiv:2607.16133](https://arxiv.org/abs/2607.16133)

### 4. Understanding Reasoning from Pretraining to Post-Training
- **方向**: Reasoning
- **摘要**: 追踪推理能力在预训练到后训练全流程中的涌现与演化，厘清各训练阶段对推理的贡献。
- **Link**: [arXiv:2607.16097](https://arxiv.org/abs/2607.16097)

### 5. DADiff: Diffusion-Driven Cross-Domain Policy Adaptation for Reinforcement Learning（IROS 2026）
- **方向**: RL / Robotics
- **摘要**: 用扩散模型驱动强化学习策略的跨域适配，缓解仿真到真实、任务间迁移的分布偏移问题。
- **Link**: [arXiv:2607.16090](https://arxiv.org/abs/2607.16090)

### 6. QUADS: Stabilizing NVFP4 Reinforcement Learning for MoE via Quantization-error Alignment across Dual Sides
- **方向**: Efficiency / 低精度 RL
- **摘要**: 通过双侧量化误差对齐，稳定 MoE 模型在 NVFP4 低精度下的强化学习训练。
- **Link**: [arXiv:2607.15810](https://arxiv.org/abs/2607.15810)

### 7. ContinuityBench: Stateful Failover in Multi-Provider LLM Routing
- **方向**: Systems / Benchmark
- **摘要**: 面向多厂商 LLM 路由的有状态故障切换基准与系统研究，评测跨供应商路由的可靠性与容错。
- **Link**: [arXiv:2607.15899](https://arxiv.org/abs/2607.15899)

### 8. When Model Merging Rivals Joint Multi-Task RL: A Task-Vector Geometry Analysis
- **方向**: ML / 模型合并
- **摘要**: 借助任务向量几何分析，揭示模型合并何时能媲美联合多任务强化学习，给出可操作的判据。
- **Link**: [arXiv:2607.16062](https://arxiv.org/abs/2607.16062)

### 9. The Honest Quorum Problem: Epistemic Byzantine Fault Tolerance for Agentic Infrastructure
- **方向**: Agent / 分布式系统
- **摘要**: 为智能体基础设施提出"诚实法定人数"问题与认知层拜占庭容错，解决多 Agent 协作中的信任与容错。
- **Link**: [arXiv:2607.16109](https://arxiv.org/abs/2607.16109)

### 10. Kolmogorov–Arnold Networks for Small Language Models
- **方向**: Efficiency / 架构
- **摘要**: 探索将 KAN（Kolmogorov–Arnold Networks）应用于小语言模型，寻求端侧/低算力下的架构增益。
- **Link**: [arXiv:2607.15525](https://arxiv.org/abs/2607.15525)
