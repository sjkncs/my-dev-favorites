# 2026-06-16 本周arXiv前沿论文

### 1. Distributed General-Purpose Agent Networks: Architecture, Key Mechanisms, and Prototypes
- **摘要**: 提出分布式通用Agent网络的架构设计，强调语义通信和信任机制在开放任务执行中的关键作用。构建了P2P Agent网络原型，使设备间可直接协作完成任务。
- **Link**: [arXiv:cs.AI](https://arxiv.org/list/cs.AI/new)

### 2. SEAGym: An Evaluation Environment for Self-Evolving LLM Agents
- **摘要**: 开发SEAGym评估环境，用于测量Agent的自我进化能力。追踪训练、验证、回放和成本记录的动态变化，为"能自我进化的Agent"提供标准化评测框架。
- **Link**: [arXiv:cs.AI](https://arxiv.org/list/cs.AI/new)

### 3. Can LLMs Be CEOs? Benchmarking Strategic Resource Reallocation with Multi-Role Agent Simulation
- **摘要**: 引入CEO-Bench评测LLM的战略资源分配能力。发现模型在多角色对抗建议场景下容易出现"单一顾问俘获"（single-advisor capture）的失败模式。
- **Link**: [arXiv:cs.AI](https://arxiv.org/list/cs.AI/new)

### 4. DeepInsight: A Unified Evaluation Infrastructure Across the Physical AI Stack
- **摘要**: 提供统一的Physical AI评测基础设施，覆盖从解码到控制的全栈评估。在保持异构性的同时实现跨层诊断可追溯性，为具身智能的标准化评测奠定基础。
- **Link**: [arXiv:cs.AI](https://arxiv.org/list/cs.AI/new)

### 5. MemTrace: Probing What Final Accuracy Misses in Long-Term Memory
- **摘要**: 通过追踪知识点而非问题行来评估记忆系统。揭示长期记忆的核心瓶颈在于**证据利用**（evidence use）而非存储容量。
- **Link**: [arXiv:cs.AI](https://arxiv.org/list/cs.AI/new)

### 6. Small Initialization Matters for Large Language Models
- **摘要**: 证明参数初始化尺度决定LLM的训练和容量。小初始化驱动低复杂度结构，反而能提升推理能力。这一反直觉发现对大模型训练实践有重要指导意义。
- **Link**: [arXiv:cs.AI](https://arxiv.org/list/cs.AI/new)

### 7. How Inference Compute Shapes Frontier LLM Evaluation
- **摘要**: 分析模型性能对推理计算预算的敏感性。提出基准测试应当将能力报告为**可用测试时资源的函数**，而非单一数字，对当前评测范式提出重要修正。
- **Link**: [arXiv:cs.AI](https://arxiv.org/list/cs.AI/new)

### 8. PreAct: Computer-Using Agents that Get Faster on Repeated Tasks
- **摘要**: 允许Agent将成功的运行轨迹"编译"为状态机程序，下次执行同类任务时先检查屏幕状态再行动，显著提升重复任务的执行速度。代表了Agent从"学会做"到"做得更快"的进化。
- **Link**: [arXiv:cs.AI](https://arxiv.org/list/cs.AI/new)
