# 2026-07-14 GitHub 热门仓库

## 1. github / spec-kit
- **描述**: Toolkit to help you get started with Spec-Driven Development — 将产品需求文档转化为可执行的 AI 代理实现计划，号称“更快构建高质量软件”。
- **Stars**: ~121k
- **Link**: [github/spec-kit](https://github.com/github/spec-kit)
- **为什么有趣**: GitHub 官方出品，聚焦“Spec-Driven Development”以对抗无序的 vibe coding；内置 `/speckit.specify`、`/speckit.plan`、`/speckit.implement` 等斜杠命令，支持 30+ AI 编码助手集成，是 AI 辅助工程化的代表性工具。

## 2. Shubhamsaboo / awesome-llm-apps
- **描述**: 100+ AI Agent & RAG apps you can actually run — clone, customize, ship。
- **Stars**: ~120k
- **Link**: [Shubhamsaboo/awesome-llm-apps](https://github.com/Shubhamsaboo/awesome-llm-apps)
- **为什么有趣**: 提供可直接运行的、Provider 无关的 AI Agent 与 RAG 模板集合，覆盖语音 Agent、多 Agent 团队、MCP Agent、LLM Memory、Generative UI 等当前热门方向，被誉为“3 条命令即可运行”的 AI 应用加速器。

## 3. Graphify-Labs / graphify
- **描述**: AI coding assistant skill，把代码、文档和媒体转成可查询的知识图谱，实现“query instead of grepping”。
- **Stars**: ~84.7k
- **Link**: [Graphify-Labs/graphify](https://github.com/Graphify-Labs/graphify)
- **为什么有趣**: Y Combinator S26 项目，支持 Claude Code、Codex、Cursor、Gemini CLI 等 20+ 编码助手；本地解析代码（无需 LLM 调用），基于 tree-sitter 提取 AST，输出交互式 HTML 报告，并与 mem0、supermemory 等做了基准对比。

## 4. moeru-ai / airi
- **描述**: Self-hosted AI companion with realtime voice chat，支持 VRM/Live2D 虚拟形象、Minecraft/Factorio 游戏互动。
- **Stars**: ~41.9k
- **Link**: [moeru-ai/airi](https://github.com/moeru-ai/airi)
- **为什么有趣**: 受 Neuro-sama 启发的自托管数字伴侣，可在浏览器和桌面运行；集成多种 LLM 提供商、实时语音合成与头像动画，代表了“AI 角色/虚拟主播 + 游戏 Agent”这一新兴赛道的工程化落地。

## 5. HKUDS / Vibe-Trading
- **描述**: Vibe-Trading: Your Personal Trading Agent — 一站式个人量化交易 Agent。
- **Stars**: ~21.8k
- **Link**: [HKUDS/Vibe-Trading](https://github.com/HKUDS/Vibe-Trading)
- **为什么有趣**: 主打“一条命令赋予 Agent 完整交易能力”，对接 18+ 市场数据源，支持回测、策略开发、Shadow Account 规则提取、印度股市和多渠道即时通讯适配器；是 AI Agent 从“写代码”走向“金融交易”这一垂直场景的热门尝试。
