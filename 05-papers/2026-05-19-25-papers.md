# 2026-05-19~25 本周arXiv前沿论文

### 1. Re-evaluating LLM Package Hallucinations on the 2026 Frontier-Model Cohort
- **arXiv**: 2605.17062
- **摘要**: 测试最新前沿LLM是否仍会幻觉不存在的软件包。发现4-6%的虚构率，识别出100+个在所有测试模型中相同的假包名，数十个仍可被恶意注册
- **类别**: 软件工程 / LLM安全

### 2. Human-Grounded Multimodal Benchmark with 900K-Scale Annotations
- **arXiv**: 2605.11663
- **摘要**: 近100万条人类标注的大规模多模态基准，评估视觉语言模型在真实世界理解任务上的能力
- **类别**: 多模态 / 基准

### 3. Source-Distributed Multimodal Agent Benchmark
- **arXiv**: 2605.15710
- **摘要**: 评估AI Agent跨分布式多模态数据源（文档/图像/数据库）推理完成复杂任务的基准
- **类别**: Agent / 多模态

### 4. LLMs Reorganize Representational Geometry During In-Context Learning
- **arXiv**: 2605.28854
- **摘要**: 研究LLM如何通过上下文示例适应新任务而不更新权重。提出ICL通过重组内部表示空间使任务相关类别更线性可分
- **类别**: LLM理论 / 可解释性

### 5. Embodied AI in Action: Insights from SAE World Congress 2026
- **arXiv**: 2605.10653
- **摘要**: SAE世界大会报告：物理AI系统在自动驾驶和制造中的真实部署洞察。强调安全关键生命周期管理和人本设计与算法突破同等重要
- **类别**: 具身智能 / 自动驾驶
