# 2026-02-05~11 GitHub Trending 热门仓库

## 1. OpenClaw
- **描述**: 本地优先的个人AI助手，支持WhatsApp/Telegram/Slack/Discord等多渠道接入，多Agent路由，语音唤醒
- **Stars**: ~210K（持续增长中）
- **URL**: https://github.com/openclaw/openclaw
- **为什么有趣**: 上周两天106K Stars的余波仍在，开发者被OpenAI挖走后项目热度不减，社区贡献者接力维护

## 2. Firecrawl
- **描述**: AI Agent的网页数据基础设施——搜索、抓取、清洗网页，输出干净Markdown
- **Stars**: ~130K
- **URL**: https://github.com/firecrawl/firecrawl
- **为什么有趣**: Agent生态的核心中间件，解决Agent"最后一公里"的数据获取问题

## 3. n8n
- **描述**: 开源工作流自动化平台，可视化界面+自定义代码+原生AI/LangChain集成
- **Stars**: Trending
- **URL**: https://github.com/n8n-io/n8n
- **为什么有趣**: Agent工作流编排的领跑者，低代码+AI原生设计让它成为非技术用户的首选

## 4. Dify
- **描述**: 生产就绪的AI Agent全链路构建平台，工作流编辑器+RAG管线管理
- **Stars**: Trending
- **URL**: https://github.com/langgenius/dify
- **为什么有趣**: 企业级Agent平台的标杆，从原型到生产的一站式体验

## 5. RAGFlow
- **描述**: 开源RAG引擎，高级检索增强生成+可追溯引用+Agent特性
- **Stars**: ~70K
- **URL**: https://github.com/infiniflow/ragflow
- **为什么有趣**: 企业知识问答的刚需工具，引用追踪功能解决了RAG幻觉的信任问题

## 6. Ollama
- **描述**: 轻量级Go框架，完全离线管理和运行大语言模型
- **Stars**: Trending
- **URL**: https://github.com/ollama/ollama
- **为什么有趣**: 本地AI运动的基石项目，OpenClaw等本地Agent项目的底层依赖
