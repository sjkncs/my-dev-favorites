# 2026-05-12~18 本周arXiv前沿论文

### 1. Agentic Discovery of Neural Architectures: AIRA-Compose and AIRA-Design
- **arXiv**: 2605.15871
- **摘要**: 多LLM协作自主发现神经架构——一个Agent负责组合模块，另一个负责设计新模块，无需人类干预
- **类别**: Agent / AutoML

### 2. Harnessing LLM Agents with Skill Programs
- **arXiv**: 2605.17734
- **摘要**: 将LLM Agent的过往经验转化为可执行的技能函数——Agent可以调用已学会的技能而非每次从头推理
- **类别**: Agent / 技能学习

### 3. Look Before You Leap: Autonomous Exploration for LLM Agents
- **arXiv**: 2605.16143
- **摘要**: 要求Agent在执行前强制进行探索阶段——先理解环境再行动，显著减少失败率
- **类别**: Agent / 探索

### 4. Delta Attention Residuals
- **arXiv**: 2605.18855
- **摘要**: 注意力机制新变体——用残差增量替代完整注意力计算，降低O(n²)到O(n)
- **类别**: LLM理论 / 架构

### 5. Reversa: Reverse Documentation Engineering for Legacy Software
- **arXiv**: 2605.18684
- **摘要**: 将遗留代码逆向工程为操作规范——为AI Agent理解和维护旧系统提供结构化文档
- **类别**: 软件工程 / Agent
