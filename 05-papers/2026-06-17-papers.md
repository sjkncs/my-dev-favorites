# 2026-06-17 本周arXiv前沿论文

### 1. Shattering the Autoregressive Curse: E³RL for LLMs
- **摘要**: 提出E³RL（Dynamic Epistemic Entropy Orchestrated Erasable RL），解决自回归推理中的级联错误问题。将不确定性锚定在交叉熵上，在保持KV缓存效率的同时切除推理缺陷。
- **Link**: [arXiv:cs.AI](https://arxiv.org/list/cs.AI/new)

### 2. FlowRAG: Synergizing Explicit Reasoning via Frequency-Aware Multi-Granularity Graph Flow
- **摘要**: 用四层异构图增强GraphRAG，通过频率感知流模块路由相关性并修剪噪声连接。在复杂推理任务上显著优于传统RAG方法。
- **Link**: [arXiv:cs.AI](https://arxiv.org/list/cs.AI/new)

### 3. FinAcumen: Financial Multimodal Reasoning via Self-Evolving Experience Memory
- **摘要**: 以选择性经验记忆库为核心的金融推理框架。通过检索过去的成功策略并抑制无关历史来提升可靠性，展示了"经验记忆"在垂直领域的实用价值。
- **Link**: [arXiv:cs.AI](https://arxiv.org/list/cs.AI/new)

### 4. STAR: SpatioTemporal Adaptive Reward Allocation for Text-to-Image RL Post-Training
- **摘要**: 在图像生成的RL后训练阶段，将奖励在图像区域间进行动态空间分配。在不改变外部条件的情况下提升文本-图像对齐和文字渲染质量。
- **Link**: [arXiv:cs.AI](https://arxiv.org/list/cs.AI/new)

### 5. Beyond Domains: Reusing Web Skills via Transferable Interaction Patterns
- **摘要**: 引入SkillMigrator，通过布局结构匹配跨域迁移Web操作技能。相比之前的工具调用系统，有效减少LLM的操作步数，代表了Agent技能复用的新范式。
- **Link**: [arXiv:cs.AI](https://arxiv.org/list/cs.AI/new)

### 6. EComAgentBench: Benchmarking Shopping Agents on Long-Horizon Tasks
- **摘要**: 为购物Agent构建长周期任务基准，要求Agent在可见查询和用户画像中发现隐藏意图。测试Agent理解"用户到底想要什么"的真实能力。
- **Link**: [arXiv:cs.AI](https://arxiv.org/list/cs.AI/new)

### 7. From Brewing to Resolution: Tracing the Internal Lifecycle of Code Reasoning in LLMs
- **摘要**: 研究LLM代码推理的内部生命周期，发现"答案酝酿"（answer brewing）和"答案确定"（resolution）两个阶段。诊断框架揭示不同Transformer架构在这两个阶段的能力差异。
- **Link**: [arXiv:cs.AI](https://arxiv.org/list/cs.AI/new)

### 8. StepGuard: Guarding Web Navigation via Single-Step Calibration
- **摘要**: 通过双策略优化和置信度引导的逐步反思来缓解Web导航中的奖励错位问题。每一步都进行校准，而非仅在最终结果层面纠错。
- **Link**: [arXiv:cs.AI](https://arxiv.org/list/cs.AI/new)
