# 2026-01-22~28 本周arXiv前沿论文

### 1. LongCat-Flash-Thinking: 560B MoE Reasoning Model
- **arXiv**: 2601.16725
- **摘要**: 美团开源560B参数MoE推理模型，统一训练框架+"Heavy Thinking"深度推理模式，专为复杂Agent推理任务设计
- **类别**: Agent / 大模型

### 2. Dynamic Role Assignment for Multi-Agent Debate
- **arXiv**: 2601.17152
- **摘要**: "Meta-Debate"框架——在多Agent辩论前通过定制化Pitch和同行评审选择最优角色分配，显著优于随机分配
- **类别**: Agent / 多智能体

### 3. MATTRL: Collaborative Multi-Agent Test-Time RL for Reasoning
- **arXiv**: 2601.09667
- **摘要**: 测试时强化学习系统，组装专门Agent进行多轮辩论和结构化记忆，绕过昂贵的预训练，在医疗/数学/教育基准上取得优异成绩
- **类别**: 推理 / 强化学习

### 4. Tokenomics: Where Tokens Are Used in Agentic Software Engineering
- **arXiv**: 2601.14470
- **摘要**: 分析多Agent编码框架的计算成本分布——迭代代码审查阶段主导Token消耗，输入Token超半数，成本主要来自持续验证而非代码生成
- **类别**: 代码生成 / Agent

### 5. InT: Self-Proposed Interventions for Credit Assignment in LLM Reasoning
- **arXiv**: 2601.14209
- **摘要**: 让模型自我评估逻辑并提出针对性修复，解决推理RL中的信用分配问题，数学基准准确率提升近14%
- **类别**: 推理 / 强化学习

### 6. EquiForm: Noise-Robust SE(3)-Equivariant Policy Learning
- **arXiv**: 2601.17486
- **摘要**: 通过几何去噪模块和对比等变对齐目标，解决3D视觉模型在传感器噪声和遮挡下的脆弱性，仿真提升17.2%，真实机器人操作提升28.1%
- **类别**: 具身智能 / 机器人

### 7. DeFM: Learning Foundation Representations from Depth for Robotics
- **arXiv**: 2601.18923
- **摘要**: 自监督框架在6000万深度图像上用DINO蒸馏+新型输入归一化策略，跨多种机器人任务迁移表现优异
- **类别**: 具身智能 / 机器人

### 8. REMAC: Real-Time Robot Execution with Masked Action Chunking
- **arXiv**: 2601.20130
- **摘要**: 通过掩码动作分块和前缀保持采样解决具身Agent的计划-执行不匹配问题，不增加计算延迟即提升任务完成率
- **类别**: 具身智能 / 机器人

### 9. Are Your Reasoning Models Reasoning or Guessing?
- **arXiv**: 2601.10679
- **摘要**: 发现层次推理模型经常"猜"而非真正推理——三种失败模式：基础假设崩溃、准确率突变、困于错误状态，提出数据集扩展等修复方案
- **类别**: LLM理论 / 可解释性

### 10. Mechanistic Understanding of Large Reasoning Models: Survey
- **arXiv**: 2601.19928
- **摘要**: 综述推理模型内部机制——学习动态、推理过程、失败模式三维度分类，桥接不透明输出到可解释内部功能
- **类别**: LLM理论 / 可解释性

### 11. QuantEval: Benchmark for Financial Quantitative Tasks in LLMs
- **arXiv**: 2601.08689
- **摘要**: 评估AI在金融三维度的能力：事实知识、数值推理、算法交易代码生成。内置回测环境执行生成代码测量实际市场表现，揭示AI与人类量化专业者的显著差距
- **类别**: 量化金融 / AI

### 12. Bayesian Noisy News Sentiment to Temporal Dynamics
- **arXiv**: 2601.16769
- **摘要**: 贝叶斯状态空间框架将噪声聚合新闻情感转化为平滑可解释时间序列，稀疏报道时自适应转向底层趋势，适用于金融信号提取
- **类别**: 量化金融 / NLP
