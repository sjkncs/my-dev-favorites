# 2026-02-18~03-03 本周arXiv前沿论文

### 1. Nabla-Reasoner: LLM Reasoning via Test-Time Gradient Descent in Latent Space
- **arXiv**: 2603.04948
- **摘要**: 推理时在潜空间做梯度下降优化文本生成，数学推理准确率提升20%+，模型调用减少10-40%。证明等价于"Deamortized PPO"
- **类别**: 推理 / 测试时计算

### 2. Memory for Autonomous LLM Agents: Mechanisms, Evaluation, and Emerging Frontiers
- **arXiv**: 2603.07670
- **摘要**: LLM Agent记忆系统综合综述——按时间跨度、数据结构和管理控制分类，提出Agent记忆的完整分类体系
- **类别**: Agent / 记忆

### 3. Improving Reasoning at Inference Time via Uncertainty-Aware Calibration
- **arXiv**: 2603.07159
- **摘要**: 解码时校准LLM推理输出的不确定性感知技术，减少过度自信和幻觉
- **类别**: 推理 / 校准

### 4. Test-Time Adaptation via Many-Shot Prompting
- **arXiv**: 2603.05829
- **摘要**: 推理时通过大量上下文示例适应模型，无需权重更新即可在新任务上获得显著提升
- **类别**: 推理 / ICL

### 5. How Large Language Models Handle Chain-of-Thought Prompts
- **arXiv**: 2603.03332
- **摘要**: Transformer内部如何处理CoT提示的机械研究——发现特定注意力头专门负责"步骤追踪"和"中间结果缓存"
- **类别**: LLM理论 / 可解释性
