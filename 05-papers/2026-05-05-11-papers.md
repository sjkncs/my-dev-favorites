# 2026-05-05~11 本周arXiv前沿论文

### 1. LLMs for Stock Price Forecasting from a Hedge-Fund Perspective
- **arXiv**: 2605.05211
- **摘要**: 综述LLM在对冲基金视角下的股价预测应用——情感提取、财报分析、价格历史Token化、多Agent框架。揭示研究中常被忽视的实操挑战（数据泄露、流动性成本、评估指标）
- **类别**: 量化金融 / LLM

### 2. Geometry Conflict: Explaining and Controlling Forgetting in LLM Continual Post-Training
- **arXiv**: 2605.09608
- **摘要**: 发现连续微调中的灾难性遗忘源于梯度更新与现有参数流形的几何冲突。提出Wasserstein度量合并技术，无需原始训练数据即可安全合并更新
- **类别**: LLM理论 / 持续学习

### 3. TRIP-Evaluate: Open Multimodal Benchmark for Transportation
- **arXiv**: 2605.00907
- **摘要**: 837项评测基准，按角色-任务-知识分类法覆盖车辆、交通管理、规划功能，跨文本/图像/点云模态。发现文本能力强但多步计算和多模态解读存在显著差距
- **类别**: 多模态 / 基准

### 4. PHBench: Predicting Series A Startup Funding Using Product Hunt Signals
- **arXiv**: 2605.02974
- **摘要**: 利用Product Hunt数据预测早期创业公司融资成功率的基准数据集，结合产品特征、社区参与度和时间模式
- **类别**: 量化金融 / ML

### 5. Cross-Modal BERT Actor-Critic for Multi-Asset Portfolio Optimization
- **arXiv**: 2605.01384
- **摘要**: Transformer Actor-Critic强化学习跨多资产类别，利用跨模态注意力捕获资产间依赖关系进行动态组合再平衡
- **类别**: 量化金融 / RL

### 6. Market Panics and Overheating via Cubic Momentum
- **arXiv**: 2605.00854
- **摘要**: 用三次动量项建模市场崩盘和过热，提供将非线性价格动态与观察到的崩盘-繁荣模式联系起来的解析框架
- **类别**: 量化金融 / 数学
