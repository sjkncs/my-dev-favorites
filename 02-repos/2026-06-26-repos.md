# 2026-06-26 GitHub Trending 开源仓库

## 1. calesthio/OpenMontage
- **描述**: Turn your AI coding assistant into a full video production studio
- **语言**: Python
- **Stars**: 约22.1k，单日新增约3.4k
- **为什么有趣**: 首个开源Agentic视频生产系统，将视频制作的12条流水线、52个工具和500+ Agent技能整合在一个仓库里，代表多Agent协作生成多媒体内容的最新方向
- **链接**: https://github.com/calesthio/OpenMontage

## 2. google-labs-code/design.md
- **描述**: 一种格式规范，用于向编程Agent描述视觉识别系统/设计系统
- **语言**: TypeScript
- **Stars**: 约19.2k，单日新增约1.5k
- **为什么有趣**: 解决Agent在代码生成中对设计系统"理解不持久"的问题，让AI在跨文件、跨会话中保持一致的设计风格，对前端/产品工程团队很实用
- **链接**: https://github.com/google-labs-code/design.md

## 3. apple/container
- **描述**: Apple官方出品的容器工具，用Swift编写并针对Apple Silicon优化，用于在Mac上创建Linux环境
- **语言**: Swift
- **Stars**: 约43.2k，单日新增约1.4k
- **为什么有趣**: 苹果罕见地进入容器/开发者基础设施领域，有望在Mac开发机上提供比Docker Desktop更原生的Linux容器体验
- **链接**: https://github.com/apple/container

## 4. aws/agent-toolkit-for-aws
- **描述**: AWS官方为AI编程Agent提供的工具包，包含MCP Server与插件，帮助Agent在AWS上构建、部署和管理应用
- **语言**: Python
- **Stars**: 约1.1k，单日新增约47
- **为什么有趣**: 云厂商直接下场做Agent工具链，标志着"Agent + 云基础设施"正成为新的标准化战场
- **链接**: https://github.com/aws/agent-toolkit-for-aws
