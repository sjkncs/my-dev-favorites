# 2026-06-23~29 本周arXiv前沿论文

### 1. Scaling Laws for Agent Architecture Design
- **arXiv**: 2606.15432
- **摘要**: 首次系统性研究Agent架构的Scaling Laws——发现Agent性能与组件数量呈幂律关系，但存在"协作瓶颈"临界点
- **类别**: Agent / 理论

### 2. Diffusion Models for Financial Time Series Generation
- **arXiv**: 2606.14891
- **摘要**: 将扩散模型应用于金融时间序列生成，能捕获传统GAN无法复制的极端事件和波动率聚集特征
- **类别**: 量化金融 / 生成模型

### 3. Mechanistic Interpretability of Transformer Attention Heads
- **arXiv**: 2606.16021
- **摘要**: 揭示Transformer注意力头的机械可解释性——发现特定头专门负责"复制""归纳""位置编码"等功能
- **类别**: LLM理论 / 可解释性

### 4. Sim-to-Real Transfer for Dexterous Manipulation
- **arXiv**: 2606.15102
- **摘要**: 灵巧操作的Sim-to-Real迁移——通过域随机化+自适应控制实现复杂抓取任务的零样本迁移
- **类别**: 具身智能 / 机器人

### 5. RLHF Alignment Tax: Quantifying the Performance Cost
- **arXiv**: 2606.14567
- **摘要**: 量化RLHF对齐的性能代价——发现对齐后模型在推理基准上平均下降3-8%，但在安全性上显著提升
- **类别**: 推理 / RL / 对齐

### 6. Multi-Agent Debate Improves Factual Accuracy
- **arXiv**: 2606.15789
- **摘要**: 多Agent辩论可显著提升LLM的事实准确性——3个Agent辩论比单一Agent减少40%的事实错误
- **类别**: Agent / 事实性
