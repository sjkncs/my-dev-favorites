# My Dev Favorites 🧠☕

> 一个程序员的" brain dump "集散地 —— 每天从信息洪流里捞出点真金白银，分类归档，定期翻出来回味。  
> 如果哪天我失业了，至少证明我曾经很努力地在学习（和摸鱼）。

## 🎯 这个仓库收录什么

### 1. 效率小技巧
- 命令行工具（Rust 系优先，毕竟快）
- IDE / 编辑器骚操作
- 一条命令解决十个问题的黑魔法

### 2. 热门开源仓库
- 近期 GitHub 上"什么鬼这么火"的项目
- 不仅看 Star 数，还看"为什么有趣"
- 从 PewDiePie 手搓的 AI 工作台到正经的企业级框架

### 3. 消费品牌 AI 工具
- 你以为只有科技公司搞 AI？错了。
- 瑞幸、喜茶、星巴克、肯德基、海底捞……全都在搞。
- 从"一句话点奶茶"到"AI 巡检后厨"，比电视剧还精彩。

### 4. 科技新闻
- AI 领域：大模型、Agent、具身智能、机器人
- 半导体/芯片：2nm 工艺、国产替代、IPO 潮
- 股市应用：量化交易、AI 投顾、监管动态

### 5. 顶会论文
- CVPR / ICLR / ICML / ACL / MLSys / NeurIPS
- 每篇配一句话摘要 + arXiv 链接
- 不装逼，看得懂才是王道

## 🤔 为什么要有这个仓库

因为互联网的信息太多，好内容太少，而人的记忆力比金鱼还短。  
与其每天刷 100 条 Twitter 然后全部忘记，不如让 AI 帮你筛选、归档、沉淀。

这个仓库的存在就是为了回答一个问题：  
**"三个月前那个看起来很酷的项目叫什么来着？"**
