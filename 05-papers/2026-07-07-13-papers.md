# 2026-07-07~13 本周arXiv前沿论文

### 1. Transformer Circuit Analysis: Mapping the Full Forward Pass
- **arXiv**: 2607.04521
- **摘要**: 首次完整映射Transformer前向传播的电路级分析——识别出负责特定推理任务的子电路，为机械可解释性提供突破
- **类别**: LLM理论 / 可解释性

### 2. RLHF vs DPO: A Comprehensive Empirical Comparison
- **arXiv**: 2607.03892
- **摘要**: RLHF与DPO的全面实证对比——发现在多数场景下DPO效果接近RLHF但训练成本低50%，但在需要精确奖励建模的任务上RLHF仍优
- **类别**: 推理 / RL / 对齐

### 3. Embodied Foundation Model for General-Purpose Robotics
- **arXiv**: 2607.05123
- **摘要**: 通用机器人基础模型——在100+仿真环境中预训练，可零样本迁移到5种不同物理机器人平台执行操作任务
- **类别**: 具身智能 / 基础模型

### 4. LLM Code Generation for Quantitative Finance
- **arXiv**: 2607.04210
- **摘要**: LLM为量化金融生成代码——在回测框架、因子挖掘、风险模型三类任务上评测，发现GPT-5在复杂金融逻辑上仍有显著缺陷
- **类别**: 量化金融 / 代码生成

### 5. Sparse Mixture of Agents for Efficient Multi-Agent Reasoning
- **arXiv**: 2607.05567
- **摘要**: 稀疏Agent混合——每次推理只激活k个最相关的Agent（类似MoE），在多Agent推理中降低80%计算成本同时保持95%性能
- **类别**: Agent / 效率

### 6. Vision-Language-Action Model with Flow Matching
- **arXiv**: 2607.04891
- **摘要**: 流匹配VLA模型——将扩散模型的流匹配技术引入视觉-语言-动作模型，实现更平滑的机器人动作生成
- **类别**: 具身智能 / VLA

### 7. Causal Discovery from Financial Time Series Using LLMs
- **arXiv**: 2607.03456
- **摘要**: 用LLM从金融时间序列中发现因果关系——将LLM的世界知识与数据驱动的因果发现结合，在股票因子因果关系识别上优于纯统计方法
- **类别**: 量化金融 / 因果推断
